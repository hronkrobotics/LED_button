# Assembly Guide

## Prototype Assembly Overview

This guide describes the modular prototype assembly approach for the LED button controller.

## Mechanical Assembly

1. Install the Cherry MX switches into the lid assembly.
2. Install the spacer behind the lid assembly.
3. Press the LED matrix holder into the switch subassembly.
4. Route the LED matrix cable through the designated openings.
5. Install the LED matrix into the holder.
6. Press-fit the diffuser cap over the LED matrix.
7. Verify smooth button travel.

## Main Body Assembly

1. Install the OLED display.
2. Install the auxiliary pushbuttons.
3. Install the battery holder or shield support.
4. Install the USB-C extension.
5. Assemble the PCB and mount the ESP32 module.
6. Connect all JST-XH connectors.
7. Remove the battery shield power switch if present and route external switch wires.
8. Mount the PCB into the enclosure.
9. Connect the OLED, LED matrix, buzzer, pushbuttons, and battery harnesses.
10. Install the batteries and close the enclosure.

## Final Checks

- All connectors seated correctly
- No pinched wires
- Buttons actuate smoothly
- LED matrix functions normally
- OLED initializes correctly
- USB charging works
- External power switch functions
