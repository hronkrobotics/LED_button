# Walnut Add-on

## Purpose

The walnut add-on is an optional accessory for the LED button platform. It is intended to provide a premium visual finish, improved tactile feel, or a custom enclosure accent while preserving compatibility with the existing electronics.

## Compatibility

- Compatible with the existing modular PCB and enclosure layout
- Intended for use with the current ESP32 firmware and button architecture
- Requires no change to core firmware behavior unless additional sensors or features are introduced

## Installation

1. Confirm the add-on part is compatible with the current enclosure geometry.
2. Install the add-on hardware using the specified fasteners.
3. Verify that switch travel and display clearance remain unchanged.
4. Re-test the device after installation.

## Required Hardware

- Fasteners specified by the add-on design
- Any spacer or bracket included with the accessory

## Firmware Compatibility

The base firmware should remain compatible. Additional firmware behavior should be added only if the add-on introduces new sensing or actuation functions.
